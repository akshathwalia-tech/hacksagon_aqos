AQOS is a comprehensive Digital Twin and AI-driven platform designed to mitigate non-revenue water (NRW) loss and optimize urban water distribution.
<br><br>
TECH STACK<br>
Backend: FastAPI (Python), Uvicorn (Asynchronous Server)<br>
Machine Learning: Scikit-Learn (Isolation Forest), TensorFlow/Keras (LSTM Forecasting)
<br>
Frontend: Vanilla JavaScript (ES6+), Chart.js (Temporal Analytics), CSS3 (Glassmorphic UI)
<br>
Data & APIs: Open-Meteo (Environmental Telemetry), Python-based Stochastic Simulator
